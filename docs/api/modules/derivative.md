# 求导

在"求导"页面可对显函数与隐函数求导，并支持代入具体数值求值。

## 功能

- **显函数求导**：输入一元函数表达式（如 `x**3 + sin(x)`），求指定阶数的导数。
- **隐函数求导**：对含 `y(x)` 的隐式方程求导。
- **代入求值**：求导结果可代入具体数值（如 `x=2`）计算导数值。
- 结果自动**分母有理化**（见 [字符串解析](../core/sympify.md#五分母有理化)）。

## 输入示例

| 含义 | 输入 |
|------|------|
| x 的三次方加正弦 | `x**3 + sin(x)` |
| 含参函数 | `a*x**2 + b*x + c` |

## 提示

- 表达式遵循 [字符串解析](../core/sympify.md) 的输入标准，也可用 `$` 前缀输入 LaTeX。
- 已定义的自定义函数（见 [字符串解析](../core/sympify.md#四函数输入标准)）可直接在求导中输入并调用，如 `f(x).diff(x)`。
