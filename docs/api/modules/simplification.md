# 表达式变形

在"表达式变形"页面提供 13 种表达式变换方法。

## 支持的方法

- 化简（simplify）
- 展开（expand）
- 因式分解（factor）
- 通分
- 部分分式（apart）
- 三角恒等变换
- 对数变换
- 换元
- 以及其它常见变形方法

## 使用方式

1. 在表达式输入框中输入待变换的表达式。
2. 从下拉框选择变形方法。
3. 点击计算，结果以 LaTeX 与纯文本两种形式显示，并自动执行分母有理化。

## 输入示例

| 含义 | 输入 |
|------|------|
| 多项式 | `x**2 - 1` |
| 三角式 | `sin(x)**2 + cos(x)**2` |

## 提示

- 表达式遵循 [字符串解析](../core/sympify.md) 的输入标准，也可用 `$` 前缀输入 LaTeX。
