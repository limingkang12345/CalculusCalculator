# 函数性质分析

在"函数性质"页面可对一元函数进行常见性质分析。

## 支持的性质

- 函数值域
- 单调区间（递增 / 递减区间）
- 奇偶性（奇函数 / 偶函数 / 非奇非偶）
- 周期性（周期函数及其最小正周期）
- 最值（最大值 / 最小值）

## 输入示例

| 含义 | 输入 |
|------|------|
| 二次函数 | `x**2 - 2*x + 1` |
| 三角函数 | `sin(x) + cos(x)` |

## 提示

- 表达式遵循 [字符串解析](../core/sympify.md) 的输入标准，也可用 `$` 前缀输入 LaTeX。
- 定义域可在"定义"页面用 `Interval` / `FiniteSet` 等限定，详见 [字符串解析](../core/sympify.md#三集合--区间输入标准)。
