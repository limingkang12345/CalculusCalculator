# 方程与不等式

在"方程""不等式""方程组""不等式组"页面求解代数方程与不等式。

## 功能

- **方程**：一元 / 多元方程求解。
- **不等式**：一元 / 多元不等式求解。
- **方程组**：多个方程联立求解。
- **不等式组**：多个不等式联立求解。
- **微分方程**：在对应选项中求解常微分方程（输入标准见 [字符串解析](../core/sympify.md#六微分方程输入标准)）。
- 结果自动**分母有理化**。

## 输入要点

- 方程用 `Eq(左式, 右式)` 或直接以 `左式 = 右式` 形式输入（具体以页面提示为准）。
- 可在"主元取值范围"中用 `FiniteSet` / `Interval` 限定只求解该范围内的根，详见 [字符串解析](../core/sympify.md#三集合--区间输入标准)。
- 已定义的自定义函数可直接参与方程，如 `f(x) = 0`。

## 微分方程示例

勾选"微分方程"，左式 `f(x).diff(x,1)`，右式 `f(x)+1`，求解得到 `f(x)=C1*exp(x)-1`。

## 提示

- 表达式遵循 [字符串解析](../core/sympify.md) 的输入标准，也可用 `$` 前缀输入 LaTeX。
