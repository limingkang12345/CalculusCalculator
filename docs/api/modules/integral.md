# 积分

在"积分"页面可进行不定积分与定积分计算。

## 功能

- **不定积分**：求原函数（结果含积分常数）。
- **定积分**：指定积分上下限，求定积分数值 / 符号结果。
- 结果自动**分母有理化**（见 [字符串解析](../core/sympify.md#五分母有理化)）。

!!! note
    v1.6.3 修复了此前不能进行不定积分和定积分的问题。

## 输入示例

| 含义 | 输入 |
|------|------|
| 不定积分 ∫x² dx | `x**2` |
| 定积分 ∫₀¹ x² dx | 表达式 `x**2`，下限 `0`，上限 `1` |

## 提示

- 表达式遵循 [字符串解析](../core/sympify.md) 的输入标准，也可用 `$` 前缀输入 LaTeX。
