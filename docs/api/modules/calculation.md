# 计算选项卡

在"计算"页面提供多种计算引擎，适合快速数值 / 符号计算与 LaTeX 生成。

## 支持的引擎

1. **Python 内置引擎**：使用 Python 内置 `eval` 快速计算数值表达式，结果为普通文本。
2. **Mpmath 高精度引擎**：可自定义十进制精度，适合高精度数值计算，结果为普通文本。
3. **SymPy 符号引擎**：基于 SymPy 进行符号运算，结果**同时以 LaTeX 代码和纯文本两种形式**显示。
4. **LaTeX 代码生成引擎**：根据输入的 Python 表达式直接生成对应的 LaTeX 代码。

## 使用方式

1. 选择计算引擎。
2. 在表达式输入框中输入表达式（支持 `$` 前缀的 LaTeX 输入）。
3. 点击计算，结果区域显示对应形式的输出。

## 与公式渲染的关系

SymPy 符号引擎与 LaTeX 生成引擎的输出会按 [公式渲染](../core/rendering.md) 中描述的方式呈现，便于复制与复用。

## 提示

- 表达式遵循 [字符串解析](../core/sympify.md) 的输入标准。
