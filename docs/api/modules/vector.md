# 向量运算（v1.5.0）

在"定义向量"页面定义二维向量并执行向量运算。

## 定义向量

- **向量名称**：默认为 `v`，可自定义。
- **向量 x/y 坐标**：分别输入 x、y 分量，支持 SymPy 表达式（如 `sqrt(3)`、`pi/2`）。
- 点击**保存**将向量加入列表，点击**删除**移除选中向量。

## 向量属性

- **表达式**：以坐标形式显示向量。
- **模**：向量长度 `sqrt(x² + y²)`。
- **方向角**：向量与 x 轴正方向的夹角。
- **单位向量**：与该向量同向、模为 1 的向量。

## 向量运算

1. 在下拉框中选择两个已定义的向量。
2. 选择运算类型：**加法**、**减法**、**点积**、**夹角**。
3. 点击**计算**，结果以 LaTeX 与纯文本两种形式显示。

!!! example
    定义 `v1 = (1, 0)`、`v2 = (0, 1)`，加法得 `(1, 1)`，点积为 `0`，夹角为 `π/2`。

## 提示

- 坐标输入遵循 [字符串解析](../core/sympify.md) 的输入标准。
